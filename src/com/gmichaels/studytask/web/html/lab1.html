Задачи и вопросы для самостоятельной работы студентов по тематике лабораторной работы:
1. Спроектируйте базу данных, состоящую из нескольких таблиц таким образом,
чтобы в многотабличной системе были таблицы со связью 1:1, 1:N.
2. Отредактируете несколько записей в базе данных.
3. Отсортируйте данные в базе данных по какому-либо признаку или полю, записав результаты сортировки.
4. Составьте вторичный индекс из одного, двух или более полей.
